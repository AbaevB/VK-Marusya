# VK-Маруся

Веб-приложение для поиска и оценки фильмов, созданное на основе Vue 3 с использованием TypeScript и Vite.

## Описание

VK-Маруся — это удобное приложение для любителей кино, которое позволяет:

- 📽️ **Просматривать случайные фильмы** — получать рекомендации случайных фильмов одним нажатием кнопки
- ⭐ **Изучать топ фильмов** — просматривать 10 лучших фильмов по рейтингу
- 🎭 **Фильтровать по жанрам** — выбирать фильмы по жанрам (боевик, комедия, драма и др.)
- ❤️ **Добавлять в избранное** — сохранять понравившиеся фильмы в личном кабинете
- 📺 **Смотреть трейлеры** — открывать трейлеры фильмов во встроенном плеере

## Функциональность

### Главная страница
- Случайный фильм с обложкой, рейтингом, описанием и кнопками действий
- Топ-10 фильмов по рейтингу
- Кнопки "Трейлер", "О фильме", "В избранное", "Другой фильм"

### Страница жанров
- Список всех доступных жанров в виде карточек
- Изображения для каждого жанра

### Страница жанра
- Фильмы, отфильтрованные по выбранному жанру
- Пагинация с кнопкой "Показать ещё"

### Страница фильма
- Обложка и постер фильма
- Рейтинг, год выпуска, жанр, длительность
- Полная информация о фильме (бюджет, выручка, режиссёр, продакшен, награды)
- Кнопки "Трейлер" и "В избранное"
- Модальное окно с видеоплеером для просмотра трейлера

### Личный кабинет
- Два раздела: "Избранные фильмы" и "Настройка аккаунта"
- Удаление фильмов из избранного
- Просмотр и редактирование профиля
- Выход из аккаунта

### Авторизация
- Форма входа и регистрации в модальном окне
- Валидация полей (email, пароль, имя, фамилия)
- Переключение между формами
- Автоматическое открытие окна авторизации при попытке добавить в избранное

## Технологии

- **Vue 3** — фреймворк для построения пользовательского интерфейса
- **TypeScript** — строгая типизация
- **Vite** — сборщик и инструмент разработки
- **Pinia** — управление состоянием
- **Vue Router** — маршрутизация
- **Axios** — HTTP-клиент для работы с API
- **SASS/SCSS** — препроцессор стилей
- **Jest** — фреймворк для юнит-тестирования

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откроет dev-сервер по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

Выполняет проверку типов и собирает оптимизированную версию в папку `dist/`

### Предпросмотр продакшен-сборки

```bash
npm run preview
```

### Запуск тестов

```bash
npm test
```

Запускает все юнит-тесты проекта.

### Запуск тестов с покрытием

```bash
npm run test:coverage
```

## Работа с API

Приложение использует внешнее API для получения данных о фильмах. Подробная документация доступна по адресу: https://cinemaguide.skillbox.cc/docs/

### Особенности авторизации

- Авторизация основана на сессиях
- При успешном входе сервер устанавливает cookie с информацией о сессии
- Для корректной работы необходимо отправлять запросы с `withCredentials: true`
- При истечении сессии требуется повторная авторизация
