# Задание

Требуется реализовать функционал формы поиска в хедере приложения.

## Правила проекта

### Стандарты кода

- Строгая типизация TypeScript
- Не создавть блоки стилей в файлах vue. Все стили уже созданы и подключены глобально через `src/assets/scss/style.scss`. При необходимости можно редактировать файлы  `src/assets/scss/components/_header.scss` и `src/assets/scss/components/_search.scss` а так же создавать новые файлы стилей в папке `src/assets/scss/components`
- При стилизации **использовать старые стандарты SASS** - с использованием @import
- Использование БЭМ для CSS селекторов обязательно.

### Архитектура

Использовать имеющиеся соответствующие папки для компонентов, тестов, моков, API, стилей, e.t.c

### Тестирование

- Создать unit-тесты для нового функционала поиска и провести тестирование.
- Минимальное покрытие: 80%

### Git

- Работа выполняется на текущей ветке *feature/search*, используется Git Flow.

- Желательно делать коммиты по стандартам **Conventional Commits**

## Описание задания

При нажатии на поле «Поиск» под полем ввода поиска должен открываться выпадающий список с результатами поиска - при вводе текста в поле запрашивается список фильмов и отображается 5 первых из полученного списка. Элемент списка содержит миниатюру постера, рейтинг, год выхода, жанр, продолжительность и название фильма. При клике на элемент открывается страница соответствующего фильма.
